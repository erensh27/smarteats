## 📋 Data Structure

Each dish contains:

```typescript
{
  name: string;                    // "Aloo Ki Sabzi"
  description: string;             // Brief description
  cookingTime: string;             // "20 minutes"
  difficulty: 'Easy' | 'Medium' | 'Hard';
  cuisine: string[];               // ["Indian", "North Indian"]
  dietaryTags: string[];           // ["Vegan", "Gluten-Free"]
  tasteTags: string[];             // ["Savory", "Spicy"]
  ingredients: string[];           // Full ingredient list
  instructions: string[];          // Step-by-step instructions
  nutrition: {
    calories: string;              // "220"
    protein: string;               // "4g"
    carbs: string;                 // "38g"
    fat: string;                   // "7g"
    fiber: string;                 // "5g"
    vitamins: {                    // Vitamin content
      "Vitamin C": "30% DV",
      "Vitamin B6": "20% DV"
    },
    minerals: {                    // Mineral content
      "Potassium": "18% DV",
      "Iron": "8% DV"
    }
  };
  portionSize: string;             // "1 serving (200g)"
  substitutions: string[];         // Healthy alternatives
  whyRecommended: string;          // Health benefits
}
```

## 🔍 Usage Examples

### Get Personalized Recommendations

```typescript
import { getPersonalizedMeals } from '@/data/masterDishDatabase';

// Example 1: Vegan, Spicy, Indian, Quick meals
const meals = getPersonalizedMeals({
  dietaryRestrictions: ['Vegan'],
  tastePreferences: ['Spicy'],
  cuisinePreferences: ['Indian'],
  maxCookingTime: 30,
  count: 5
});

// Example 2: High-protein, Low-carb meals
const proteinMeals = getPersonalizedMeals({
  dietaryRestrictions: ['High-Protein', 'Low-Carb'],
  minProtein: 25,
  calorieRange: { min: 300, max: 500 },
  count: 3
});

// Example 3: Quick vegetarian Indian breakfast
const breakfast = getPersonalizedMeals({
  dietaryRestrictions: ['Vegetarian'],
  cuisinePreferences: ['Indian'],
  maxCookingTime: 20,
  difficulty: 'Easy',
  count: 3
});
```

### Filter by Specific Criteria

```typescript
import {
  getVegetarianIndianDishes,
  getHighProteinDishes,
  getLowCalorieDishes,
  getQuickMeals,
  getWeightLossDishes,
  getMuscleBuildingDishes
} from '@/data/masterDishDatabase';

// Get all vegetarian Indian dishes
const vegIndian = getVegetarianIndianDishes();

// Get high-protein dishes (20g+)
const highProtein = getHighProteinDishes();

// Get quick meals (under 20 minutes)
const quick = getQuickMeals();

// Get weight loss friendly dishes
const weightLoss = getWeightLossDishes();

// Get muscle building dishes (25g+ protein)
const muscleBuilding = getMuscleBuildingDishes();
```

### Search Dishes

```typescript
import { searchDishes } from '@/data/masterDishDatabase';

// Search by name or description
const results = searchDishes('paneer');
// Returns all paneer dishes

const dals = searchDishes('dal');
// Returns all dal varieties
```
